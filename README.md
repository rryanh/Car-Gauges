## Getting Started

To get a local copy up and running follow these example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/rryanh/Car-Gauges.Git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
